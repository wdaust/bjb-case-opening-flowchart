Medical Records — Process Overview

Records Request  |  Submission  |  Follow-Up  |  QA Review

The Medical Records department handles provider record requests from initial review through final submission. Each task follows an escalating SLA structure with standard and complex timelines.

Pre-Send Verification Checklist

1. Provider Information (You Already Have This — Keep It)
  • Provider name
  • Correct facility/location
  • Phone number
  • Fax number
  • Email (if applicable)
  • Department (Medical Records / Billing / Radiology, etc.)
2. HIPAA Authorization Verification (Critical)
  • Correct patient name
  • Date of birth
  • SSN (if required by provider)
  • Signature present
  • Signature date valid (within provider timeframe)
  • Witness required (if applicable)
  • Expiration date valid
  • Scope of authorization (records + billing if needed)
3. Request Scope Verification (Often Missed — Very Important)
  • Dates of treatment specified
  • Records vs. Billing vs. Films clearly identified
  • Itemized billing requested (if applicable)
  • Radiology images requested (if applicable)
  • EMS records requested (if applicable)
  • Prior records requested (if applicable)
4. Case Information Verification
  • Correct client name
  • Correct date of loss
  • Correct claim number (if applicable)
  • Correct attorney / firm contact information
  • Correct return method (fax/email/mail)
5. Special Provider Requirements (High Value Check)
  • Provider-specific authorization form required?
  • Copy of ID required?
  • Payment required upfront?
  • Online portal submission required?
6. Duplicate Request Prevention (Operational Excellence)
  • Verify request has not already been sent
  • Verify records not already received
  • Check prior follow-up notes
7. Quality Control / Compliance Check
  • HIPAA properly attached
  • Request letter attached
  • All pages legible
  • No missing documents
  • Correct provider selected
